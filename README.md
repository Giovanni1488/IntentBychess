Разработчики: Бычков Даниил Кириллович и Дмитриев Захар Андреевич
Описание: Приложение представляет собой одноэкранную «Контактную книгу», которая позволяет звонить, отправлять email, просматривать адрес на карте и делиться контактными данными через соответствующие системные приложения с использованием механизмов Intent.
